# limen-cli

Optional admin command-line interface for [Limen](https://github.com/getlimen/limen).

**Status: not yet started.** Planned for v1.1+ — the v1 admin surface is the Angular UI.

## Planned commands

```
limen login
limen node list
limen node add
limen service deploy <name>
limen route add <hostname>
limen backup create
limen logs <service> --follow
```
